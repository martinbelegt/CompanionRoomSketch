# Companion RoomSketch Foundation v1

Versie: 0.1

Status: Foundation

---

# Missie

Companion RoomSketch is een interactieve woningplanner.

De applicatie maakt het mogelijk om woningen op schaal in te richten met echte meubels.

De nadruk ligt op eenvoud, snelheid en nauwkeurigheid.

---

# Filosofie

Niet tekenen.

Maar bouwen.

Alles binnen de applicatie is een object.

Een muur is een object.

Een deur is een object.

Een bank is een object.

Een plant is een object.

Een complete woning is een verzameling objecten.

---

# Ontwerpprincipes

De Engine kent geen meubels.

De Engine kent alleen objecten.

Objecten kennen hun eigenschappen.

De UI presenteert objecten.

Businesslogica hoort nooit in de UI.

---

# Belangrijkste doel

Versie 1 moet Yvette in staat stellen zelfstandig het appartement in Hank volledig digitaal in te richten.

Daarna wordt Companion RoomSketch uitbreidbaar voor iedere woning.
